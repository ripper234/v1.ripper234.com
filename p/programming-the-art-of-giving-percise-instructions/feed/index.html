Note that the biggest number will be at most (2X)!/(X!)^{2}, (by setting X=Y). So there is no need to calculate the number (X+Y)! to its bitter end. Probably the most efficient way to calculate this asymptotically, is by using Stirling’s formula. Note that by the nature of this approximation, it is evident that the complexity is way beyond exponential. It is in fact something like X^{X}.
