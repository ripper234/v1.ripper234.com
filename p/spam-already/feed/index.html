After some experimentation, I came to the conclusion that a captcha’s popularity makes it vulnerable. And indeed, the standard captcha of WordPress was broken by spammers. I have a hunch this is the case for Blogger as well.

So I made a custom “captcha” on my blog. While completely idiotic, it stops spammers dead because I’m probably the only one using it so no one is spending resources to devise targeted algorithms to break it (not that you need any…)
